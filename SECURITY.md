# Security Policy

## Public Portfolio Demonstration

This repository is a public static dashboard demo using synthetic data. It is not connected to production systems and should not receive confidential, client, employer, or personal information.

## Secrets

Do not commit API keys, tokens, passwords, private keys, or environment-specific credentials.

## Responsible Disclosure

If you find an exposed secret or a security issue in this demo, please report it through the repository owner rather than opening a public issue with sensitive details.
